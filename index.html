CopperLauncher

CopperLauncher is an application that launches Minecraft: Java Edition on your Android devices with additional convenience features such as instance-based game data separation, easy installation of various mod loaders and modpacks, advanced (enough) on-screen controls and more.

The official CopperLauncher pages and communities are:

  • The GitHub page
  • The International Discord server
  • The YouTube channel

You can download CopperLauncher for Android from the following places:

  1. Download from GitHub Actions. This is where in-development versions are released. GitHub sign-in is required.
  2. Build it yourself from source

CopperLauncher is currently not available on iOS devices. If you want this to change, you should support the project so that iOS development becomes possible.

After obtaining the app, you should learn how to get started.
