Contributing to the Wiki

This page covers how to write and edit wiki content specifically. For general contributing info, see Contributing.

What Can I Contribute?

  • Fixing typos or outdated information
  • Adding missing steps or clarifications
  • Writing new guides for undocumented features
  • Adding or updating screenshots

Writing Good Wiki Pages

Headers

Use <h1> for page title, <h2> for major sections, <h3> for subsections.

Tips and Warnings

Use the tip, warning, or danger CSS classes on a <div> for callout boxes — as seen throughout this wiki.

Need Help?

Ask on Discord or open a GitHub issue.
