Shaders

Shaders are demanding. Even high-end Android devices may struggle. Start with lightweight shader packs.

Compatibility

RendererShader Support
GL4ESLimited — some shaders work
MobileGluesBetter compatibility with modern shaders
VirGLVaries

Recommended Shader Packs for Mobile

  • Complementary Unbound (Low preset) — Beautiful lighting, reasonably optimized
  • Sildur's Vibrant Shaders Lite — Lightweight, works on many devices
  • BSL Shaders (Low preset) — Good balance of quality and performance

Installing Shaders (Iris — Fabric)

  1. Install Fabric + Iris Shaders.
  2. Launch Minecraft.
  3. Go to Options → Video Settings → Shader Packs.
  4. Open the shader packs folder, copy your .zip file in.
  5. Select the shader.

Performance Tips for Shaders

  • Use the Low or Lite preset if available.
  • Reduce render distance to 4–6 chunks.
  • Disable or reduce shadow distance in shader settings.
  • Use Sodium alongside Iris for best Fabric performance.

If shaders cause crashes, try switching the renderer in Copper Launcher settings (GL4ES → MobileGlues or vice versa).
