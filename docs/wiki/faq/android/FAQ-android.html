FAQ — Android

Installation

Where do I download Copper Launcher?

  • Stable: GitHub Releases
  • Nightly: Nightly.link

Why does Android block the installation?

You need to allow installation from unknown sources. See the Install guide.

Is Copper Launcher on the Play Store?

Not currently. Install via APK from GitHub.

Performance

The game runs slowly. What can I do?

See the Optimization Guide. Quick tips: lower render distance to 4–6 chunks, set Graphics to Fast, and allocate the right amount of RAM.

How much RAM should I allocate?

Device RAMRecommended
3GB1–1.5GB
4GB1.5–2GB
6GB2–3GB
8GB+3–4GB

My device gets very hot. Is that normal?

Minecraft is CPU/GPU intensive. Some heat is expected. Enable Sustained Performance mode in launcher settings to reduce heat.

Crashes & Errors

The game crashes on launch. What do I do?

  1. Check the crash log (tap Show Log in launcher settings).
  2. Search GitHub Issues for your error.
  3. Open a new issue with the full log attached if it's a new bug.

"Java runtime not found" error

Go to Settings → Java Runtime and re-download the runtime for your architecture.

"OpenGL error" or black screen

Try switching the renderer in Settings → Renderer (e.g. GL4ES → MobileGlues).

Mods

Can I use mods with Copper Launcher?

Yes! See the Mods guide.

Which mod loaders are supported?

Forge and Fabric are both supported.
