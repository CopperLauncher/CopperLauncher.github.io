Custom Controls

Copper Launcher features a fully rewritten controls system built in native code, giving you a responsive and customizable experience.

Opening the Controls Editor

While in-game, tap the gamepad icon in the top-right corner to open the controls editor. Alternatively, go to Settings → Controls → Edit Controls from the launcher home screen.

Control Types

Virtual Joystick

The left joystick controls movement. You can resize it, move it anywhere on screen, and toggle fixed vs. floating mode in control settings.

Buttons

Buttons can be added, moved, resized, and removed. Available actions include Jump, Sneak, Sprint, Attack, Use Item, Open Inventory, Drop Item, Chat, Perspective Toggle, and any keyboard key.

Mouse Mode

Enable Mouse Mode in the controls settings for a trackpad-style experience, useful for some Minecraft versions.

Importing / Exporting Controls

  1. Open the controls editor.
  2. Tap the Export button to save your layout as a .json file.
  3. To import, tap Import and select a .json controls file.

Platform Tips

Bluetooth controllers and USB OTG keyboards/mice are supported on Android. Copper Launcher uses standard Android gamepad mapping.

Resetting Controls

Open the controls editor → tap the menu icon (three dots) → select Reset to Default.
