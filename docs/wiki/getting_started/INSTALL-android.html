Installing Copper Launcher (Android)

Requirements

  • Android 8.0 (Oreo) or higher
  • At least 500MB free storage
  • A Minecraft: Java Edition account (or local/offline account)

Copper Launcher supports ARM32, ARM64, x86, and x86_64. Most modern Android phones are ARM64.

Step 1 — Download the APK

StableGitHub ReleasesRecommended for most users
NightlyNightly.linkLatest features, may be unstable

Step 2 — Allow Unknown Sources

  1. Open Settings on your device.
  2. Go to Apps (or Security, depending on your device).
  3. Tap Install unknown apps.
  4. Select your browser or file manager and enable Allow from this source.

Remember to disable this setting after installation for security.

Step 3 — Install the APK

  1. Open your Downloads folder.
  2. Tap the APK file.
  3. Tap Install when prompted.
  4. Tap Open to launch Copper Launcher.

Step 4 — First Launch

On first launch, Copper Launcher will ask for storage permissions and download the Java runtime for your device. Continue to Your First Launch to finish setup.

Troubleshooting

Installation blocked? Make sure you've enabled unknown sources (Step 2).

App crashes on launch? Try the latest nightly build, or report the issue on GitHub Issues.

Not enough storage? Free up at least 1–2GB for a smooth experience.
